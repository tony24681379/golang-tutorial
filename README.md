# The Introduction of Go

## Prerequire

- Install Golang
- Install present package

```
$ go get golang.org/x/tools/present
$ go install golang.org/x/tools/present
```

### Demo

```
$ present
2017/11/30 14:28:55 Open your web browser and visit http://127.0.0.1:3999
```
